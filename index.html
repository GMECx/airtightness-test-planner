Airtightness Test Planner — Help & About

Airtightness Test Planner is a free, single-file tool for preparing whole-building airtightness test plans from construction PDFs. Open a PDF: mark up the tested boundary, blower-door locations, interior and exterior reference tubes, temporary seal locations, and any notes or RFIs, then export a clean plan as a PNG or PDF.

Getting started

Click Open PDF (or drop a file anywhere on the workspace). Multi-page PDFs load every page — use the ◀ ▶ arrows or PageUp/PageDown to move between them. Each page keeps its own annotations and undo history.

Drawing tools (left toolbar)

Select & Move Click to select; drag to reposition; double-click to edit text on manometers, callouts, and RFIs. Drag empty space to pan.
Tested Boundary Click to add a boundary around the pressurized volume. Return the line to the first point (or double-click) to close the polygon. Closed boundaries stay highlighted in orange.
Blower Door Fan Multi-point red line. Drag near a manometer port to snap to a channel.
Exterior Reference Green line indicating an exterior envelope reference tube.
Interior Reference Blue line with arrowhead indicating an interior envelope reference tube.
Temp Seal Area Click & drag to draw a dashed ellipse marking a temporary seal (door, vent, damper, etc.).
Manometer Places a dual-channel unit with four ports (A-Input, A-Ref, B-Input, B-Ref). Enter its serial number. Connected lines follow the manometer when moved.
Text Callout Adds a yellow note with a leader line. Drag the dot to relocate the leader endpoint.
RFI / Issue Tag Adds a red tag for open questions or field conditions that need clarification.
Place Key / Legend Drops a KEY legend box onto the plan, auto-populated with the symbols you've used. Drag it to wherever fits best.

Keyboard shortcuts

Shift (hold)Lock your in-progress drawing line to 0° / 90° from last point.
Space (hold)Pan the view while dragging.
Scroll wheelZoom in / out, centered on the cursor.
EscCancel an in-progress line or boundary.
Delete / BackspaceWhile drawing: remove the last point placed. Otherwise: delete the selected item.
Ctrl+Z / Ctrl+YUndo / Redo (per page).
PageUp / PageDownPrevious / next page.

Exporting

Export PNG saves the current page as a high-resolution image with the auto-generated KEY legend. Export PDF writes every page into a single PDF file, ready to share with the client, GC, or field crew.
